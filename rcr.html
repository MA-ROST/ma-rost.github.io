UI System - Core

One of our restrictions on this project was that we needed to be able to play our game without the use of a keyboard or mouse. We quickly discovered that Unreal's UI does not natively support controller to UI interaction (in a way that is understandable to players, anyways). The alternative solution, using the existing plugin, CommonUI, was interfering with the core of our game, so that was also not an option.
What our solution ended up being was to create a new system functionally similar to CommonUI, but entirely based in blueprints. Using core concepts from Our largest challenge when it came to Rift City Rebels was that natively, Unreal does not allow a controller to communicate with UI as smoothly as other methods.
